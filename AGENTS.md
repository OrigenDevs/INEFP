# Reglas del Proyecto INEFP

## Comunicacion
- SIEMPRE usar el lector de voz (mensaje.txt) para dar feedback al usuario
- NUNCA usar el question panel (question tool) - el usuario lo tiene prohibido porque le atrasa el proyecto
- Confirmar cada accion por el lector de voz

## Git
- No preguntar antes de commitear, simplemente hacerlo y avisar por lector de voz
- No tocar archivos de Juan (rama origin/Juan) sin que el usuario lo pida
- Verificar cambios de Juan con git fetch periodicamente

## Codigo
- NUNCA usar UnityEngine.Input, siempre usar UnityEngine.InputSystem
- No tocar codigo VR existente (VRHandController, SimpleGrab, etc.) a menos que el usuario lo pida
- Si el usuario dice que algo funciona en VR, no intentar "arreglarlo"
- Proyecto Unity 6 en C:\DAVID\UNITY\INEFP
- Branch master, repo: https://github.com/OrigenDevs/INEFP
